目的：创建持久化的终端会话（不会由于 ssh 断开导致运行停止），断开后可重新连接

```{bash}
# 新建tumux会话 
tmux new -s my_session 

# 激活环境并运行脚本 
conda activate 你的环境名 

python 你的脚本.py  
Rscript 你的脚本.R 

脱离 tumux 会话：按下 ctrl+B，然后按 D 

# 查看有哪些会话在运行 

tmux ls
```

# 重新连接会话

```{bash}
tmux attach -t my_session
```

# 杀死 session

```{bash}
tmux kill-session -t my_session 
```

**如何移动光标查看历史记录：按下 ctrl+B，松开，然后按 \[，即可进入复制模式，**

**按 q 退出**

# zellij

类似的工具有很多，比较新的有zellij，可以使用zellij -h查看帮助文档，感觉比tmux用起来方便些，而且几乎完全兼容tmux的快捷键，无缝切换
