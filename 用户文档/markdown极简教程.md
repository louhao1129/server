# 📝 Markdown 极简入门指南

## 什么是 Markdown？

Markdown 是一种轻量级的标记语言。 简单来说，它让你**像写纯文本一样写文档**，不需要复杂的排版工具（像 Word 那样），就能自动生成格式精美的文章。

**使用场景：** Github 的 `README` 文件、Jupyter Notebook 的说明单元格、技术博客、服务器文档。

------------------------------------------------------------------------

## 1. 标题 (Headings)

使用 `#` 号来表示标题，`#` 越多，标题越小。

**输入代码：**

``` markdown
# 是一级标题
## 是二级标题
### 是三级标题
#### 是四级标题
```

**实际效果：** 

# 是一级标题 

## 是二级标题 

### 是三级标题

------------------------------------------------------------------------

## 2. 文本样式 (Text Styling)

让文字**加粗**、*倾斜*或~~删除~~。

**输入代码：**

``` markdown
**这是加粗的文字**
*这是倾斜的文字*
~~这是删除线~~
`这是强调（行内代码）`
```

**实际效果：** **这是加粗的文字** *这是倾斜的文字* ~~这是删除线~~ `这是强调（行内代码）`

------------------------------------------------------------------------

## 3. 代码块 (Code Blocks) 🔥 **最重要的功能**

对于我们服务器用户来说，这是必学的！用来展示命令或脚本。 使用三个反引号 \`\`\` 包裹代码。

**输入代码：** \`\`\`bash \# 这是一个代码块，可以指定语言（如 bash, python） sudo apt update echo "Hello World" \`\`\`

**实际效果：**

``` bash
# 这是一个代码块，可以指定语言（如 bash, python）
sudo apt update
echo "Hello World"
```

你也可以写 Python 代码： \`\`\`python import numpy as np print("Markdown is awesome!") \`\`\`

**实际效果：**

``` python
import numpy as np
print("Markdown is awesome!")
```

## 4. 列表 (Lists)

整理思路必备。

**输入代码：**

``` markdown
**无序列表（前面加 - 或 *）：**
- 苹果
- 香蕉
- 橘子

**有序列表（前面加 1.）：**
1. 第一步：打开冰箱
2. 第二步：把大象装进去
3. 第三步：关上冰箱
```

**实际效果：** 

**无序列表：** 
- 苹果 
- 香蕉 
- 橘子

**有序列表：** 
1. 第一步：打开冰箱 
2. 第二步：把大象装进去 
3. 第三步：关上冰箱

## 5. 链接与图片 (Links & Images)

**输入代码：**

``` markdown
[点击这里访问百度](https://www.baidu.com)

![图片加载失败时显示的文字](图片链接.jpg)
```

**实际效果：** [点击这里访问百度](https://www.baidu.com) *(图片会直接显示在这里)*

> **记忆口诀：** 链接是 `[中括号](圆括号)`。图片只是在前面多加一个感叹号 `!`。

------------------------------------------------------------------------

## 6. 引用 (Blockquotes)

用于解释说明或警告。

**输入代码：**

``` markdown
> 这是一段引用文本。
> 它可以用来做提示信息。
```

**实际效果：** \> 这是一段引用文本。 \> 它可以用来做提示信息。

------------------------------------------------------------------------

## 💡 给服务器用户的建议

1.  **Jupyter Notebook:** 在 Cell 类型里选择 "Markdown"，按 `Shift + Enter` 渲染，用来给你的代码写注释。
2.  **VS Code:** 打开 `.md` 文件后，点击右上角的预览按钮（或者按 `Ctrl + K` 然后按 `V`），可以实时预览效果。
3.  **README.md:** 任何项目的根目录下都应该有一个 README.md，用来告诉别人怎么运行你的代码。
