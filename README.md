第一次使用数信云服务器必看： 《新用户使用手册》 [http://sxygptcloud.com:7000](http://sxygptcloud.com:7000/)

1.  先从浏览器中打开sxycloud1.top:5000网页，使用账号密码登录控制台，获取服务器登录信息。

2.  登陆网页之后，ssh终端用户名为控制台账号，ssh终端密码为控制台的默认密码。

3.  查看服务器状态需要为running，如果是Stopped,手动点击开机。

4.  在【SSH连接】查看ssh终端的登录信息，注意使用显示的ssh端口。

5.  Rstudio、Jupyter服务可直接从控制台中点击网址跳转打开。
