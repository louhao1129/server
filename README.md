**---常见问题---**

**直接关闭浏览器标签页 ≠ 退出程序**。

在 VHPC 这种共享服务器上，如果不正确退出，你的后台进程（R 语言引擎或 Python 内核）会一直卡在服务器内存里，**持续占用 CPU/显存**，甚至导致下次登录时报错。

[正确退出网页版Rstudio和Jupyter](用户文档/正确退出网页版Rstudio和Jupyter.md)

***Rstudio-server使用后记得及时保存结果并清理内存，否则下一次打开可能会打不开!***

Rstudio-server退出后再登录一直转圈，打不开: 因为你没有正常退出Rstudio，正确流程应该是先点击右上角的红色按钮退出，再关闭网页 现在只能1.先关闭网页 2.登录终端，执行Rclean 3.重新登录网页

**Tips**

在终端运行的命令可以使用Ctrl+c进行强制终止
